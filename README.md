# Insurance Analytics Project

## Project Overview

This project focuses on analysing performance and business KPIs using interactive dashboards built in Excel, Power BI, and Tableau.
The KPIs were calculated using SQL queries, and the dashboards provide insights into revenue performance, achievements, opportunity pipeline, and invoice distribution.

The goal of this project is to demonstrate data analysis, data modeling, and visualization skills using industry-standard tools.

---

## Tools & Technologies

* Excel – Dashboard creation, Pivot Tables, KPI cards, slicers
* Power BI – Interactive dashboard and data modeling
* Tableau – Visual analytics and dashboard design
* SQL – KPI calculations and data aggregation

---

## Key Performance Indicators (KPIs)

The following KPIs were calculated and visualized:

* Cross Sell PLCD Achievement %
* Cross Sell Invoice Achievement %
* New Sell PLCD Achievement %
* New Sell Invoice Achievement %
* Renewal Sell PLCD Achievement %
* Renewal Sell Invoice Achievement
* No. of Invoices by Account Executive
* Top Open Opportunities
* Opportunity Pipeline Funnel
* Opportunity distribution by Product
* Yearly Meeting Count
* No. of Meetings by Account Executive
* Average deal size by Product
* Cross Sell (Achieved, Target, New)
* New (Achieved, Target, New)
* Renewal (Achieved, Target, New)
* Total Opportunity 
* Total Open Opportunity
